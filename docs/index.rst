Jira one
=================
A REST API Implementation to Jira Cloud APIs for creating reports and for performing other Jira queries.


Configurations
-------
Install using ``pip``. you have to be on python >= 3.6.x in order to utilize this script.

.. code-block:: bash

    pip install jiraone
